# Demo: Ejemplo Demo

## Orden rápido de edición

1. **`brief.md`** — investigación del prospecto
2. **`js/config.js`** — nombre, contacto, WhatsApp, nav
3. **`css/variables.css`** — colores (o copiar preset de `_shared/color-presets.json`)
4. **`index.html`** — textos de secciones + meta title/description
5. **`assets/images/`** — reemplazar `hero.svg` y `about.svg` (o subir fotos reales)
6. **`robots.txt`** y **`sitemap.xml`** — URL de la demo publicada

## Vista local

Abrí `index.html` en el navegador o usá Live Server.

## Publicar

Subí la carpeta completa a tu hosting (Netlify, Vercel, cPanel, etc.) y actualizá `status.json` con la URL.
