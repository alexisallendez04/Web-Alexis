# Checklist por demo

Usá esto en cada carpeta de prospecto. Tiempo objetivo: **1–2 horas** por demo.

## Antes de codear (15 min)

- [ ] Google Maps / Instagram revisados
- [ ] `brief.md` completado
- [ ] Paleta elegida (`_shared/color-presets.json`)

## Personalización (45–60 min)

- [ ] `js/config.js` — nombre, contacto, WhatsApp
- [ ] `css/variables.css` — colores de marca
- [ ] `index.html` — hero, especialidades, nosotros, FAQ
- [ ] `assets/images/hero` — imagen representativa
- [ ] `assets/images/about` — foto o ilustración
- [ ] Favicon si hay logo del prospecto

## Antes de publicar (10 min)

- [ ] Title y meta description correctos
- [ ] Links de WhatsApp y teléfono funcionan
- [ ] Vista mobile OK
- [ ] `robots.txt` y `sitemap.xml` con URL real
- [ ] `status.json` → `"estado": "publicada"` + `url_demo`

## Prospección

- [ ] Demo subida al hosting
- [ ] Mensaje personalizado (`_shared/outreach/mensaje-template.md`)
- [ ] `status.json` → `"estado": "contactada"` + fecha en `contactado`
- [ ] Entrada actualizada en `registry.json`

## Si cierra el cliente

- [ ] Mover o copiar a `demos/` como proyecto real
- [ ] Agregar al portfolio principal cuando esté en producción
